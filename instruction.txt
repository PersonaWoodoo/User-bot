📚 Инструкция по использованию команд бота

Этот файл содержит подробное описание всех команд бота и примеры их использования.

1. 🔍 <b>.dox</b>
   - Описание: Генерация случайного докса (вымышленной информации о человеке).
   - Пример:
     <code>.dox</code>
     Бот сгенерирует случайный докс и отправит его вам.
     Пример вывода:
     ```
     📱
     ├ Номер: 1234567890
     ├ Страна: Россия
     ├ Регион: Москва
     └ Оператор: МТС

     📓 Имя: Иван
     🏠 Адрес: ул. Ленина, 1
     🏥 Дата рождения: 01.01.1990

     👁 Интересовались: 50 человек.
     🏅 Репутация: 100👍 10👎

     💬 Социальная метка: Активный пользователь (5)
     👁 Комментариев: 200
     ```

2. ➕ <b>.add [ключ] [значение]</b>
   - Описание: Добавить данные в базу для генерации докса.
   - Примеры:
     <code>.add number 1234567890</code> - Добавить номер телефона.
     <code>.add country Россия</code> - Добавить страну.
     <code>.add region Москва</code> - Добавить регион.
     <code>.add oper МТС</code> - Добавить оператора.
     <code>.add names Иван</code> - Добавить имя.
     <code>.add adres ул. Ленина, 1</code> - Добавить адрес.
     <code>.add birth 01.01.1990</code> - Добавить дату рождения.
     <code>.add goodrep 100</code> - Добавить лайки.
     <code>.add badrep 10</code> - Добавить дизлайки.
     <code>.add interest 50</code> - Добавить количество интересующихся.
     <code>.add social Активный пользователь 5</code> - Добавить социальную метку.
     <code>.add comment 200</code> - Добавить количество комментариев.
   - Пример:
     <code>.add number 1234567890</code>
     Вывод: <code>✅ 1234567890 добавлено в numbers.</code>

3. 🎭 <b>.troll</b>
   - Описание: Начать троллить пользователя (ответьте на сообщение).
   - Пример:
     <code>.troll</code> (ответьте на сообщение пользователя)
     Бот начнет отправлять случайные комплименты этому пользователю.
     Пример вывода:
     ```
     ✅ Теперь я буду отправлять комплименты пользователю 123456789!
     ```
     После этого бот будет отправлять случайные комплименты из списка.

4. 🤖 <b>.gpt [запрос]</b>
   - Описание: Запрос к GPT-4 для генерации текста.
   - Пример:
     <code>.gpt Напиши рассказ про космос</code>
     Бот отправит ответ, сгенерированный GPT-4.
     Пример вывода:
     ```
     🤖 Ответ GPT-4:
     В далекой галактике, где звезды светят ярче, чем на Земле, жил космический путешественник по имени Зорг...
     ```

5. 🖼 <b>.img [запрос]</b>
   - Описание: Генерация изображения по запросу.
   - Пример:
     <code>.img Космический корабль</code>
     Бот отправит сгенерированное изображение.
     Пример вывода:
     ```
     ⏳ Генерация изображения...
     ✅ Изображение отправлено.
     ```

6. 💬 <b>.addtroll [комплимент]</b>
   - Описание: Добавить комплимент для троллинга.
   - Пример:
     <code>.addtroll Ты самый лучший!</code>
     Бот добавит этот комплимент в список для троллинга.
     Пример вывода:
     ```
     ✅ Комплимент добавлен: Ты самый лучший!
     ```

7. 📨 <b>.anon [Тема], [Сообщение], [Почта получателя]</b>
   - Описание: Отправить анонимное сообщение на указанную почту.
   - Пример:
     <code>.anon Привет, Это тест, example@gmail.com</code>
     Бот отправит сообщение с указанной темой и текстом на почту example@gmail.com.
     Пример вывода:
     ```
     ✅ Сообщение отправлено с почты: example1@gmail.com
     ```

8. 🎤 <b>.tts [текст]</b>
   - Описание: Преобразовать текст в голосовое сообщение.
   - Пример:
     <code>.tts Привет, как дела?</code>
     Бот отправит голосовое сообщение с этим текстом.
     Пример вывода:
     ```
     ✅ Голосовое сообщение отправлено.
     ```

9. 📋 <b>.menu</b>
   - Описание: Показать меню всех доступных команд.
   - Пример:
     <code>.menu</code>
     Бот отправит список всех команд с описанием.
     Пример вывода:
     ```
     📜 Доступные команды:

     🔍 .dox - Генерация случайного докса.
     ➕ .add [ключ] [значение] - Добавить данные в базу.
     🎭 .troll - Начать троллить пользователя (ответьте на сообщение).
     🤖 .gpt [запрос] - Запрос к GPT-4.
     🖼 .img [запрос] - Генерация изображения по запросу.
     💬 .addtroll [комплимент] - Добавить комплимент для троллинга.
     📨 .anon [Тема], [Сообщение], [Почта получателя] - Отправить анонимное сообщение.
     🎤 .tts [текст] - Преобразовать текст в голосовое сообщение.
     📋 .menu - Показать это меню.
     ```

10. 📝 <b>Дополнительные инструкции</b>
    - Убедитесь, что файл emails.txt содержит почты и пароли в формате:
      <code>"example@gmail.com" "pass123"</code>
    - Для работы команды .anon почты должны поддерживать SMTP (например, Gmail).
    - Для работы команды .gpt требуется доступ к GPT-4 через API.

📌 Примечание:
- Все команды работают только от вашего аккаунта.
- Для команд, требующих ответа на сообщение (например, .troll), обязательно отвечайте на сообщение пользователя.
